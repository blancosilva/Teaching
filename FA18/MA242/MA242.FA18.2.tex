y''+y'-2y=e^x
r^2+r-2=0
r=1,-2
y_1 = e^x        y'_1 = e^x          y''_1 = e^x
y_2 = e^{-2x}    y'_2 = -2e^{-2x}    y''_2 = 4e^{-2x}
y = Ae^x + Be^{-2x} + y_p(x)
y_p(x) = Cxe^x because s=1
y'_p(x) = Ce^x + Cxe^x
y''_p(x) = 2Ce^x + Cxe^x

y = Ae^x + Be^{-2x} + Cxe^x
y' = Ae^x -2Be^{-2x} + Ce^x + Cxe^x
y'' = Ae^x +4Be^{-2x} + 2Ce^x + Cxe^x

   Ae^x + 4Be^{-2x} + 2Ce^x +  Cxe^x
   Ae^x - 2Be^{-2x} +  Ce^x +  Cxe^x
 -2Ae^x - 2Be^{-2x}         - 2Cxe^x
======================================   
                      3Ce^x

                      
It has to be 3Ce^x = e^x; therefore, C=1/3.  The general solution is:
y = Ae^x + Be^{-2x} + (1/3)xe^x

Now, for the initial conditions:
y(0)  =   0:     A+B=0
y'(0) = 4/3:     A-2B+1/3=4/3

Solution: (A=1/3, B=-1/3)

y = \tfrac{1}{3}\big( e^x - e^{-2x} + xe^x \big)


With Laplace:

y''+y'-2y = e^x
s^2 F - 4/3 + sF - 2F = (s-1)^{-1}
(s^2+s-2)F = (s-1)^{-1} +4/3

F = (s-1)^{-1} (s-1)^{-1} (s+2)^{-1} + (4/3) (s-1)^{-1} (s+2)^{-1}
