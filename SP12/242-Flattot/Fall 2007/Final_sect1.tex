\noindent \textbf{\large Problem}\\[0.2cm]
We consider the initial value problem
\begin{equation}\label{diff_eq}
y''-3y'+2y=20x\cos(2x),
\end{equation}
with the initial values $y(0)=0$, $y'(0)=1$.\\
We want to solve it by two different ways.\\[0.2cm]
\textbf{ I : Laplace transform}\\
\hspace*{0.2cm} $1)$ Use the theorem of differentiation of transforms to find the Laplace transform of $x\cos(2x)$.\\[4cm]
\hspace*{0.2cm} $2)$ Let $\ds{K(s)=20\frac{s+2}{(s-1)(s^2+4)^2}}$. Show that
\[ K(s)=\frac{12}{5}\frac{1}{s-1}+\frac{-12s+8}{(s^2+4)^2}+\frac{1}{5}\frac{-12s-12}{s^2+4}. \]
\newpage
\noindent \hspace*{0.2cm} $3)$ Then solve the initial value problem using Laplace transform.\\[12cm]
\textbf{ II : Classical way}\\
\hspace*{0.2cm} $1)$ Find the complementary solution of (\ref{diff_eq}).\\[3cm]
\hspace*{0.2cm} $2)$ Use the sheet about the particular solution to find one for (\ref{diff_eq}).\\
\vspace*{13cm} \;\\
\hspace*{0.2cm} $3)$ Then solve the initial value problem.\\[6cm]
\textbf{ III : Conclusion } What method do you prefer ?\\
